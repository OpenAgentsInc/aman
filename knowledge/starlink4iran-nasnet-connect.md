# NASNET Connect (Starlink4Iran)
Source: https://connect.starlink4iran.com/en/
Category: Connectivity and networking
Organization: NASNET Connect

## Overview
A service focused on configuring Starlink connections with MikroTik routers. It emphasizes secure sharing, VPN support, and multi-WAN routing.

## Key features
- Supports multiple router models and VPN protocols.
- Multi-WAN routing with failover and load balancing.
- Geographic traffic optimization for domestic and foreign links.

## Example uses
- Combine Starlink with a local ISP and set automatic failover.
- Separate domestic and international traffic for performance and policy needs.
- Build a shared access point with VPN-protected routing.

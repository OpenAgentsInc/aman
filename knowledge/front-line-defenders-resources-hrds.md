# Front Line Defenders Resources for HRDs
Source: https://www.frontlinedefenders.org/en/resources-hrds
Category: Human rights defender resources
Organization: Front Line Defenders

## Overview
A library of handbooks, training manuals, and workbooks for human rights defenders at risk.

## Key resources listed
- Workbook on Security with practical steps for HRDs.
- Trial Observation Handbook for monitoring court proceedings.
- Protection Handbook for dealing with threats and intimidation.
- Guides on UN field presences and EU support.
- Gendering Documentation Manual for documenting violations.

## Example uses
- Run a security planning workshop using the workbook.
- Train volunteers to observe trials and report due process issues.
- Build a protection plan aligned with UN or EU support channels.

## Notes
- The resources emphasize systematic risk assessment and practical tactics.

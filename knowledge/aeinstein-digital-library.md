# AEI Digital Library (Albert Einstein Institution)
Source: https://www.aeinstein.org/digital-library
Category: Nonviolent action library
Organization: Albert Einstein Institution

## Overview
A digital library of nonviolent action resources with extensive translation options.

## Key titles listed
- 198 Methods of Nonviolent Action
- From Dictatorship to Democracy (text and audio)
- Civilian-Based Defense
- The Anti-Coup
- Exploring Nonviolent Alternatives

## Example uses
- Build a reading list for movement training.
- Distribute translations suited to local audiences.
- Use specific titles as workshop modules.

## Notes
- Many titles are available in multiple languages.
